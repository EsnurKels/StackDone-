# StackDone

StackDone, modern web teknolojileriyle geliştirilmiş, kullanıcı dostu ve performans odaklı bir yapılacaklar listesi (To-Do List) uygulamasıdır. 

**Canlı Önizleme:** [https://stack-done.vercel.app/](https://stack-done.vercel.app/)

## Özellikler

- Görev ekleme, silme ve tamamlama işlemleri.
- Görevler için öncelik seviyeleri (Düşük, Orta, Yüksek).
- Modern ve temiz kullanıcı arayüzü.
- Hızlı çalışma ve düşük yüklenme süreleri.

## Kullanılan Teknolojiler

Bu proje, güncel frontend geliştirme araçları kullanılarak inşa edilmiştir:

- **React:** Bileşen tabanlı kullanıcı arayüzü kütüphanesi.
- **Vite:** Hızlı geliştirme ortamı ve yapılandırma aracı.
- **TypeScript:** Tip güvenliği ve daha sürdürülebilir kod yapısı.
- **Tailwind CSS:** Modern ve hızlı arayüz tasarımı için yardımcı sınıf (utility-first) CSS çerçevesi.
